# Polyparasitism in Intestinal Schistosomiasis, Malawi

## Data and code repository for intestinal schistosomiasis study

The data required to run the [analysis](https://amoreo71.github.io/malawi_intestinal_schisto/malawi_intestinal_schisto_V2.html) in R can be found within the 'data' folder.

This work has been published in *Microbial Genomics*:

*Angus M. O'Ferrall, Lucas J. Cunningham, David Lally, Peter Makaula, Gladys Namacha, Alexandra Juhász, Sam Jones Ruth Cowlishaw, Sarah Rollason, Priscilla Chammudzi, Donales R. Kapira, Sekeleghe A. Kayuni, E. James LaCourse, Adam P. Roberts, Janelisa Musaya, J. Russell Stothard*

[**Dynamic interactions between Schistosoma haematobium, Schistosoma mattheei and Schistosoma mansoni underscore the complex polyparasitism of intestinal schistosomiasis in southern Malawi**](https://royalsocietypublishing.org/rstb/article/381/1941/20250191/478997/Dynamic-interactions-between-Schistosoma)
